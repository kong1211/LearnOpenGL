# 操作说明

## 灯光选择
- 按下 **1**, **2**, **3** 键可以选择不同的灯光（灯光 1、灯光 2 或灯光 3）。

## 模型选择
- 按下 **7** 键选择模型 1。
- 按下 **8** 键选择模型 2。
- 按下 **9** 键选择模型 3。
- 按下 **0** 键选择模型 4。

## 显示/隐藏模型
- 按下 **4** 键切换模型 1 的显示与隐藏。
- 按下 **5** 键切换模型 2 的显示与隐藏。
- 按下 **6** 键切换模型 3 的显示与隐藏。
- 按下 **7** 键切换模型 4 的显示与隐藏。

## 模型移动
- 按下 **U** 键模型将沿 **Y 轴** 向上移动。
- 按下 **J** 键模型将沿 **Y 轴** 向下移动。
- 按下 **I** 键模型将沿 **Z 轴** 向前移动。
- 按下 **K** 键模型将沿 **Z 轴** 向后移动。
- 按下 **O** 键模型将沿 **X 轴** 向右移动。
- 按下 **L** 键模型将沿 **X 轴** 向左移动。

## 相机切换
- 按下 **C** 键切换相机类型（模型查看相机和飞行相机）。  
  （首次按下时切换，释放后可以再次切换。）

## 旋转状态切换
- 按下 **P** 键切换模型的旋转状态（开启或关闭旋转）。

## 退出窗口
- 按下 **ESC** 键关闭窗口。

## 相机移动
- 按下 **W** 键相机前进。
- 按下 **S** 键相机后退。
- 按下 **A** 键相机左移。
- 按下 **D** 键相机右移。

## 改变光照颜色
- 按下 **B** 键切换灯光颜色。
